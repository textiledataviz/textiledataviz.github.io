Erica Lord's [[The Codes We Carry]] replicates computer-produced DNA/RNA microarrays from diseases disproportionately affecting Indigenous communities using beadwork. Adapting existing visualizations where the data has gone through a computer-based processing pipeline is one approach; taking data at an earlier stage (e.g. directly from instruments, or after some basic level of processing) and creating a visualization using the conventions of the textile you're working in is another.
