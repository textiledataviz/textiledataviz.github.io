[Sonja Boon](https://sonjaboon.com/)'s project [Stitching Empire](https://stitchingempire.wordpress.com/) includes several days that draw directly on data. [Day 20: Stitchery](https://stitchingempire.wordpress.com/2025/02/02/day-20-stitchery/) represents the classification of 50 indentured laborers by skin color, using [[Surface Embroidery]] to depict [[Historical Data]]. This piece uses thread color as its property.
