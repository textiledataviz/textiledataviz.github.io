Rebecca Elder used [[Textiles/Weaving/index|weaving]] to create a hand towel with a pattern that uses the dashes and dots of Morse Code [to spell out "Eat a bag of dicks"](https://bsky.app/profile/elvissa.bsky.social/post/3lhmqp5lytk2b). The text itself, translated into Morse Code and then woven into in the fabric of the towel, is the data ([[Data from Text]]).
